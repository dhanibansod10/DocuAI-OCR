#  DocuAI – AI Powered OCR

##  Features
- Extract text from images (OCR)
- Grammar correction using NLP
- Clean UI with React

##  Tech Stack
- Frontend: React, Vite, Tailwind CSS
- Backend: Flask, OpenCV, Tesseract OCR

## How to Run

### Backend
cd backend
python app.py

### Frontend
cd frontend
npm install
npm run dev
